## Share Block

### JSON Repair

![](./static/1.png)

Fix JSON formatting errors

**Input Parameters**:

- `content`: JSON content to be repaired
- `indent`: Output indentation, set to `0` for single-line output
- `preview`: Whether to enable preview

**Output**

- `output`: Repaired JSON content
