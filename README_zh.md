## Share Block

### JSON Repair

![](./static/1.png)

修复 JSON 格式错误

**输入参数**:

- `content`: 需要修复的 JSON 内容
- `indent`: 输出的缩进，为 `0` 时则输出为一行
- `preview`: 是否开启预览

**输出**

- `output`: 修复后的 JSON 内容
